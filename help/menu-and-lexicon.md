# 菜单与用户词库

本文只讲候选框菜单里普通用户最常用的功能，尤其是用户词库维护流程。

## 设置菜单

“设置”里现在只保留普通用户可直接理解的功能：

- 候选列表：每页个数、横排/竖排显示
- 反查信息：控制汉字反查时显示哪些内容
- 交互：当前唤起热键、修改热键、唤起方式、休眠方式
- 外观：前景颜色、背景颜色、字体大小、主界面透明度、活动窗始终置顶

这些设置改动后会自动持久化，不需要再手工编辑设置文件。

### 反查信息怎么选

“设置 -> 反查信息”里现在有这几种模式：

- 默认（推荐：标准拼音 + 音元拼音）：适合大多数日常查看场景
- 完整（标准拼音 + 数字标调 + 音元拼音 + 键位序列）：适合排查、教学或对照检查
- 隐藏反查信息（音元拼音熟练者可选）：只保留候选和状态
- 仅标准拼音：只看带声调的普通拼音
- 仅音元拼音：只看音元编码
- 仅键位序列：只看实际需要敲的键位

如果你只关心“这个字词该敲哪几个键”，可以直接选“仅键位序列”。

## 用户词库菜单

“工具 -> 用户词库”里现在分成两组入口：

- 添加当前词条
- 删除当前词条
- 编辑与重载：编辑用户词库、应用用户词库
- 导入与导出：导入用户词库、导出用户词库

## 推荐使用流程

### 本机维护

1. 用“编辑用户词库”生成并打开可编辑文本；这一步会写出固定文件 `user_lexicon_import.txt`。
2. 修改并保存后，用“应用用户词库”把这份文件重新读回当前环境。

### 外部导入

如果词库来自另一台机器、备份文件或外部整理结果：

1. 准备 UTF-8 文本，并放到同一个固定文件名 `user_lexicon_import.txt`。
2. 用“导入用户词库”写入当前环境。

### 备份导出

如果想备份当前词库：

1. 用“导出用户词库”导出当前内容；这一步会写出 `user_lexicon_export.txt`。
2. 再自行归档或复制到别处。
