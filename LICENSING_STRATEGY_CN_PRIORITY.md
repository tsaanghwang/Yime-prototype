# 境内优先授权文本方案

本文件给出一套以中国境内主体、中文沟通和境内争议解决为优先的授权文本组织方案，适用于主要用户、客户和执法场景均集中在中国境内的项目。

## 适用情形

建议在下列情形优先采用本方案：

1. 主要用户、潜在客户和合作方位于中国境内。
2. 商业许可证谈判、签署和履行主要以中文完成。
3. 预期争议解决、证据固定和执法行动主要发生在中国境内。

## 文本组织建议

1. 根许可可继续以英文为主文本，同时保留一份完整中文参考译文。
2. 商业合同模板应提供中文主版本，并在需要时附英文对照。
3. 商业说明、FAQ 和授权申请材料优先采用中文，再附精简英文说明。

## 建议在商业合同中明确的条款方向

1. 适用法律：中华人民共和国法律。
2. 争议解决：许可方所在地有管辖权的人民法院，或双方约定的中国境内仲裁机构。
3. 通知方式：电子邮件、书面快递和合同列明的联系人地址。
4. 违约处理：停止使用、删除副本、停止分发、补签授权或承担违约责任。

## 建议采用的对外说明口径

可在商务邮件、网站或 README 中使用如下正式表述：

> 本项目默认允许非商用使用。任何涉及销售、收费服务、客户交付、闭源集成、SaaS 托管或其他营利性业务活动的使用，均须事先取得单独书面商业许可证。商业许可证的适用范围、期限、费用、支持内容及争议解决方式，以双方签署的商业许可证为准。

## 优点

1. 便于与中国境内客户、律师和法务团队沟通。
2. 条款表达、通知方式和争议解决路径更贴合境内实践。
3. 有利于快速形成中文商业合同和正式函件。

## 风险与注意事项

1. 对国际用户的可读性和接受度相对较弱。
2. 若未来主要面向海外客户，可能需要另行准备更完整的英文合同体系。
3. 若第三方材料来自境外，还需单独核对其跨法域许可兼容性。
