# 使用说明（USAGE.md）

本手册详细介绍 YIME 音元输入法编辑器的主要功能、界面操作、常见流程和快捷键，帮助用户快速上手和高效使用。

## 1. 启动与访问

- 启动 Web 原型：
  ```bash
  npm run start
  ```
- 访问地址： http://localhost:3000

## 2. 主界面说明

- 输入区：用于输入拼音、音元或汉字。
- 码表区：显示候选汉字、词语及其编码。
- 设置区：切换输入模式、键盘布局、主题等。
- 用户词库管理：添加、删除自定义词条。

## 3. 常见操作流程

- 切换输入模式：在界面设置区选择全拼、简拼、双拼或并击。
- 输入汉字：按音元或拼音输入，候选区选择目标汉字。
- 添加用户词：在词库管理区输入拼音和词语，点击“添加”。
- 切换键盘布局：设置区选择美式/中式键盘。

## 4. 快捷键

- 回车/空格：确认输入
- 上/下方向键：切换候选项
- Esc：取消输入
