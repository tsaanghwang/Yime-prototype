# 快速开始

本文只讲第一次把当前外挂输入法原型跑起来并试用的最短路径。

## 启动方式

推荐入口：

`python -m yime.input_method.app`

等价入口：

`python run_input_method.py`

常用参数：

- `--copy-only`：只复制候选，不自动回贴。
- `--font-family "Microsoft YaHei"`：指定候选框字体。

## 启动后看到什么算正常

正常情况下，启动后可能看到：

- `键盘监听已启动，按ESC退出`
- `键盘监听未启用，将使用手动输入模式`
- `[Decoder] 运行时候选已回退到 SQLite 数据库视图 runtime_candidates`

这些都不一定表示出错，要结合环境和启动日志一起判断。

## 基本操作

- 直接在外部窗口输入码元，候选框会在光标附近出现。
- 首选候选可用 `Space`、`Enter` 或鼠标左键直接选择。
- 第 2~5 候选可依次用 `` ` ``、`-`、`=`、`\` 选择，也可鼠标左键直接点击。
- 任意候选都可先用方向键移动高亮，再按 `Space` / `Enter` 确认；也可直接鼠标左键点击。
- 如果每页候选数设为 6~9，第 6~9 候选建议用鼠标左键，或先用方向键移动高亮后再按 `Space` / `Enter`。
- `Home` / `PgUp` / `PgDn` / `End` 可翻页。
- `ESC` 用于退出当前组合，`Ctrl+Q` 用于关闭窗口。
- 待命状态下，可点“音”图标或按当前唤起热键恢复。

## 试用建议

- 先在记事本测试。
- 再到 VS Code 普通文本区测试。
- 不要一开始就在权限复杂或快捷键很多的应用里测。
