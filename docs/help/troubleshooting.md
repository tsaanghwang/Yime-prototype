# 故障排查

本文只保留普通试用者最常见的几个问题。

## 提示“将使用手动输入模式”

先检查：

- Python 3.12 是否可用
- `pywin32` 是否可用

这通常表示当前没有进入完整的全局监听路径，但不一定意味着完全不能用。

## 候选词为空或结果不完整

先不要立刻认定为安装失败。

当前候选有多级回退：

- 运行时 JSON
- SQLite `runtime_candidates` 视图
- 静态候选表

优先看启动日志确认当前实际使用的是哪一层。

## 某些应用里行为不稳定

这是当前原型阶段的已知边界之一。

先回到下面这些简单环境复现：

- 记事本
- VS Code 普通文本区
- 其他标准输入框

如果简单环境正常、目标应用异常，更可能是目标应用兼容性差异，不一定是安装问题。

## 现在是不是正式系统输入法

不是。

当前实现是“桌面原型 + 全局钩子 + 浮动候选框”，不是系统级 TSF / IMM32 输入法。
