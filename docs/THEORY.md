# 音元输入法理论简介（THEORY.md）

## 1. 音元系统概述

音元输入法（YIME）以“音元”为最小编码单位，区别于传统以音节或拼音为基础的输入法。当前语义注册表包含 57 个 Yinyuan ID：24 个噪音类（`N01`—`N24`）和 33 个乐音类（`M01`—`M33`）。这里的类别按编码中的区别特征定义，不等同于普通声学意义上的“无音高／有音高”；详见 [噪音类与乐音类：分类说明](ZAOYIN_YUEYIN_CLASSIFICATION.md)。

- **噪音**：无固定音高，充当音节首音，对应传统声母。
- **乐音**：有固定音高，构成干音（呼音+韵音），对应传统韵母及声调。

## 2. 编码原则与结构

- **一音一码**：每个音元对应唯一键位，实现高效输入。
- **等长编码**：每个音节由 1 个噪音类 Yinyuan ID 和 3 个乐音类 Yinyuan ID 组成；首音／干音是结构段，不能与两类音元互换命名。
- **多模式支持**：等长、变长和省键模式已经进入工程主线；双拼、并击仍是设计方向。

## 3. 键盘映射与输入逻辑

- **美式键盘重置布局**：52个音元映射到标准键盘的下档/上档键位。
- **中式扩展键盘**：支持更多自定义和兼容性。
- **键音映射灵活**：可根据用户需求调整音元与键位的对应关系。

## 4. 系统优势

- **平均码长最短**：同等条件下，字词平均码长优于现有全拼输入法。
- **输入效率高**：拣字、拣词率高，减少按键次数。
- **音系表达精细**：音元系统能更准确反映汉语语音结构，便于语言学分析和扩展。

## 5. 与传统拼音/音位系统的区别

- 传统拼音以音节为单位，音元法以更细粒度的音元为单位。
- 音元系统能表达更多语音细节，支持更灵活的输入和编码。
