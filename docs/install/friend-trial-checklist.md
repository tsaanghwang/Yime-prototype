# 朋友试装前最小验收清单

这份清单只服务一个目标：

- 在把安装包发给朋友之前，用最短时间排掉最容易翻车的问题。

适用对象：

- 当前这版 `Setup.exe`
- 第一次给外部朋友试装
- 只追求“能装、能开、能基本输入、出问题时能说清楚”

## 发包前准备

1. 确认你准备发的是最新安装包，例如 `dist/setup/Yime-Setup-0.1.0.exe`。
2. 最好先卸掉本机旧安装版，避免把升级残留误当成新包正常行为。
3. 准备一个最简单的外部输入目标，例如记事本。

## 必做 6 项

1. 双击安装包，确认安装向导能完整走完，没有报缺文件、权限错误或启动失败。
2. 安装完成后，从开始菜单启动，确认右下角待命图标能出现。
3. 在记事本里测试一次 hotkey 唤起，确认候选框能弹出，不是只装上但完全叫不出来。
4. 实际上屏一到两个词，确认不是只能显示候选框，真正回贴也能成功。
5. 加一条用户词库，确认 `%LOCALAPPDATA%/Yime/user_lexicon.db` 已生成，重开应用后这条词还能用。
6. 关闭应用再重新启动一次，确认不是“首次启动正常，第二次就坏”。

## 建议再看 3 项

1. 卸载一次，确认应用目录能被移除，开始菜单入口不会残留成死链接。
2. 安装后如果创建了桌面快捷方式，点一次确认它也能正常启动。
3. 如果你准备让朋友重点试 hotkey 模式，最好先在你自己机器上确认没有和常用快捷键明显冲突。

## 发给朋友时建议一起说清楚

1. 这还是“Windows 输入法原型应用”，不是系统级输入法产品。
2. 当前最希望他反馈的是：装不装得上、能不能启动、hotkey 能不能唤起、能不能正常上屏。
3. 如果出问题，优先让他描述“卡在安装、启动、唤起、上屏、重启后失效”中的哪一步。

## 可以先不拦的事

- 没有数字签名
- 还不是系统级输入法注册
- 还没有自动更新

这些都是真的缺口，但不属于“第一次发给朋友试装前必须先补完”的门槛。
