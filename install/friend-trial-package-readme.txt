Yime 朋友试装交付包

这是当前默认的最小试用交付目录，直接用于“发给朋友试装”这一场景。

你通常只需要按这个顺序处理：
1. 先自己看 FRIEND-TRIAL-CHECKLIST.md，确认这版安装包值得发出去。
2. 再把目录里的 Yime-Setup-*.exe 发给试用者。
3. 发包时可直接参考 FRIEND-TRIAL-MESSAGE-TEMPLATE.md 改写一段短说明。
4. 如对方需要最小操作说明，一起附上 FRIEND-TRIAL-START-HERE.md。

本目录包含：
- Yime-Setup-*.exe：发给试用者的安装包
- FRIEND-TRIAL-START-HERE.md：给试用者看的最小操作说明
- FRIEND-TRIAL-CHECKLIST.md：发包前自己先过一遍的验收清单
- FRIEND-TRIAL-MESSAGE-TEMPLATE.md：发给朋友时可直接改写的短消息模板
- README.txt：本说明

说明：
- 这还是 Windows 输入法原型应用，不是系统级正式输入法产品。
- 当前最优先验证的是：能不能安装、能不能启动、hotkey 能不能唤起、能不能正常上屏。
