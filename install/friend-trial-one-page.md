# 试用者一页说明与最小键位对照

这份文档只服务一个目标：

- 给第一次试用这版安装包的人一页就够用的说明。

它不是完整理论文档，也不是全量键盘布局规范。

## 这是什么

- 这是一个可安装的 Windows 输入法原型应用。
- 它还不是系统级正式输入法产品。
- 当前最值得试的是：能不能启动、能不能唤起、能不能正常上屏。

## 怎么开始

1. 安装后，从开始菜单启动 `Yime`。
2. 先打开记事本做测试，不要一上来就在快捷键很多的软件里试。
3. 正常待命时，右下角会有一个小的待命图标。

## 怎么唤起和休眠

- 默认唤起热键：`Ctrl+Alt+Insert`
- 默认也可以点右下角的“音”图标唤起
- 默认休眠方式：再次按 `Ctrl+Alt+Insert`，或右键候选框

如果你按了热键但完全没反应，先不要怀疑码表，先把它当成“唤起失败”来反馈。

## 最常用的几个键

- `Space`：提交首选候选或提交待上屏文本
- `Enter`：提交首选候选或提交待上屏文本
- `` ` ``、`-`、`=`、`\\`：依次选择第 2~5 候选
- 方向键：移动候选高亮，之后可按 `Space` / `Enter` 确认
- 鼠标左键：可直接点击候选
- `Backspace`：删掉当前组合里的最后一个码元
- `Esc`：退出当前组合，或结束这次运行

## 最小试用流程

1. 打开记事本。
2. 按 `Ctrl+Alt+Insert` 唤起。
3. 输入几下码元，确认候选框能出来。
4. 用 `Space` / `Enter` 选首选，或用 `` ` ``、`-`、`=`、`\\`、方向键加 `Space` / `Enter`、鼠标左键选一个候选，确认能上屏。
5. 再试一次关闭后重开，确认不是只有第一次能用。

## 如果你只想给我最短反馈

直接告诉我下面哪一种最接近：

- 装不上
- 能装但打不开
- 能打开但唤不起候选框
- 候选框能出来但不能上屏
- 第一次能用，重开后失效
- 基本能用，但某个键位或手感很怪
